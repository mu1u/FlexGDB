获取主机信息 ​

FlexGDB顶部的菜单栏提供了软件注册的功能。通过复制主机信息按钮，可以快速获取主机信息。

找到软件注册功能按钮 ​

找到图示所在功能按钮，打开会弹出一个软件许可的对话框。

复制主机信息 ​

在软件许可对话框，按下复制主机信息即可成功复制主机信息。

获取注册许可 ​

通过官网填入主机信息购买许可或将主机信息发送给软件代理商获取许可。

完成注册 ​

获取许可后将许可使用Ctrl+V的方式将许可输入到上述对话框中并点击右上角的注册按钮即可完成注册：

注意事项

所有操作均需在10秒钟内完成，否则软件将会强制退出。
